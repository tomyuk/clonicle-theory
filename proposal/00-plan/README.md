# Brainful Alliance White Paper
## Proposal 執筆方針

## このディレクトリの目的

`proposal/00-plan/` は、Brainful Alliance White Paper の設計、編集、執筆、レビュー、公開方針を管理する場所です。ここに置かれる文書は完成原稿ではなく、各章をどのような思想で構成し、どの順序で理論を積み上げるかを明確にするための Proposal です。

本成果物はAIの解説書でも、DXの実践書でもありません。Chronicle Theory に基づいて企業を時間的・関係的・意味的な存在として捉え直し、そこから Brainful、Brainful Protocol、Brainful Alliance、社会実装へ至る提案体系を構築します。技術は目的ではなく、その思想を実装する手段として扱います。

## 基本原則

本書は、存在論、定義、理論、方法、Protocol、Alliance、社会実装の順に論理を積み上げます。読者へ固定的な結論を押し付けるのではなく、より良い理論と実践を共創するための提案として記述します。

文体・論理構成・用語・図表・RDE差異検証の詳細は、上位の `proposal/AGENTS.md` に従います。

## Proposal としての位置付け

本成果物は完成した教義ではありません。社会への提案として公開し、議論、批判、実践、検証、改訂を通じて発展させます。したがって、この文書群そのものも Chronicle Theory のクロニクルとして継続的に更新されます。
